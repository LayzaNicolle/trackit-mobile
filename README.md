📝 Descrição do Projeto

O TrackIt Mobile é o aplicativo que consome a API RESTful do TrackIt Back-end. Desenvolvido em React Native com a plataforma Expo, ele oferece uma interface fluida e intuitiva para que os usuários realizem o controle, visualização e rastreamento de itens e objetos emprestados em tempo real, utilizando armazenamento de estado global para sessão e cache dinâmico para as requisições.

🛠️ Tecnologias Utilizadas

- Framework: React Native + Expo
- Navegação: Expo Router
- Gerenciamento de Estado Global: Zustand
- Consumo de API & Cache: Axios + TanStack Query (React Query)
